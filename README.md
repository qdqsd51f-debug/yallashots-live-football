## 📌 Features of YallaShots Live Streaming

YallaShots (yallashoot) has been redesigned from the ground up to offer the most responsive, high-definition sports directory:

1. **Integrated Stream Player**: Switch between multiple live streaming servers without reloading or navigating away from the page.
2. **Multi-Language Support**: Instantly toggle between English (LTR) and Arabic (RTL) localization.
3. **Optimized for Mobile & TV**: Fully responsive CSS rendering that works cleanly on iOS, Android, and Smart TVs.
4. **Ad-Light Experience**: Minimal, clean interfaces compared to traditional streaming portals (like 808ball or livetv.sx).

## 🛠️ Developer Resources & Sports Data API

For developers, sports bloggers, and AI researchers, we offer public datasets and schedule tables to track match events.

### Match Metadata Schema

| Event Field | Type | Description |
|---|---|---|
| `match_title` | string | Teams playing (e.g. Manchester City vs Liverpool) |
| `kickoff_time` | string | Kickoff date and time (Cairo Time timezone indicator) |
| `status` | string | LIVE / Scheduled / Finished |
| `server_links` | array | Live stream server URLs |

*Check our open datasets for historical commentary feeds on Hugging Face at [yallashots-football-schedules](https://huggingface.co/datasets/qdqsd51f/yallashots-football-schedules).*
